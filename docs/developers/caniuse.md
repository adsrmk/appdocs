# Ondersteuning / compatibiliteitstabellen

[Can I Use↗](https://caniuse.com/) biedt actuele browserondersteuningstabellen voor front-end-webtechnologieën, voor zowel desktop- als mobiele browsers. 

Hiermee kun je snel zien welke functies veilig te gebruiken zijn, welke vendor prefixes vereisen en welke fallbacks nodig hebben voor oudere browsers — onmisbaar voor een consistente gebruikerservaring op alle platforms.
