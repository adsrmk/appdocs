# Accessibility Checker

[Accessibility Checker↗](https://AccessibilityChecker.org/) is een platform voor het testen en waarborgen van webtoegankelijkheid. Het scant je site op problemen die gebruikers met een beperking kunnen hinderen en toetst je site aan standaarden zoals **WCAG**, **ADA**, **Section 508**, **AODA** en de **European Accessibility Act**.
