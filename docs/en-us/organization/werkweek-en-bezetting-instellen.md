# Configure the workweek and staffing target

Define working days, default hours and desired availability.

<p>Organization workweek settings are used in time, attendance and leave calculations.</p>
<h2>Define working days</h2>
<p>Select the days that normally count as working days for the organization. Employee-specific agreements can differ, but the organization setting provides the default.</p>
<h2>Set expected hours</h2>
<p>Use realistic default hours. Review existing registrations after a change to make sure reports are interpreted using the correct period and settings.</p>
<h2>Set the staffing target</h2>
<p>Choose the minimum desired team availability. The target helps identify periods where approved or requested leave may create understaffing.</p>
<div class="callout callout-warning"><strong>Review before saving</strong><p>Changes can affect future calculations and how managers interpret dashboards. Communicate material policy changes to the team.</p>
</div>
