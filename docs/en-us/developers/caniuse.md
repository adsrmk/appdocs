# Support / Compatibility Tables

[Can I Use↗](https://caniuse.com/) provides up-to-date browser support tables for front-end web technologies, across both desktop and mobile browsers. It helps you quickly check which features are safe to use, which require vendor prefixes, and which need fallbacks for older browsers — essential for ensuring a consistent experience across platforms.
