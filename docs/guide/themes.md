# Thema's

In het tabblad **Thema's** beheer je de visuele identiteit van je website. Net als bij het onderdeel Plugins, kun je via dit dashboard thema's activeren, verwijderen, bijwerken (automatisch of handmatig) en wisselen — zonder dat je hoeft in te loggen op het WordPress-adminpaneel.
