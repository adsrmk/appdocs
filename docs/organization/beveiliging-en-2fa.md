# Je ondernemersaccount beveiligen

Bescherm je beheeraccount met tweestapsverificatie.

<p>Tweestapsverificatie voegt naast je wachtwoord een tijdelijke code uit een authenticator-app toe aan het inloggen.</p>
        <h2 id="wachtwoord-controleren">Begin met een sterk wachtwoord</h2>
        <p>Gebruik voor je beheeraccount een uniek wachtwoord van minimaal acht tekens. Wijzig het via Beveiliging door eerst je huidige wachtwoord in te vullen.</p>
        <h2 id="inschakelen">Tweestapsverificatie inschakelen</h2>
        <ol class="docs-steps">
          <li><div>Open de beveiligingsinstellingen van je account.</div></li>
          <li><div>Scan de QR-code met Google Authenticator, 1Password of een andere TOTP-app.</div></li>
          <li><div>Vul de gegenereerde zescijferige code in.</div></li>
          <li><div>Kies <strong>2FA inschakelen</strong> om de koppeling te bevestigen.</div></li>
        </ol>
        <h2 id="qr-niet-scannen">Kun je de QR-code niet scannen?</h2>
        <p>Open de handmatige sleutel onder de QR-code en voeg deze toe in je authenticator-app. Vul daarna de actuele zescijferige code in.</p>
        <h2 id="uitschakelen">Tweestapsverificatie uitschakelen</h2>
        <p>Wanneer 2FA actief is, staat bij Beveiliging de actie <strong>Uitschakelen</strong>. Gebruik deze alleen wanneer je direct een nieuwe beveiligingsmethode instelt.</p>
        <div class="callout callout-info"><strong>Beheerrechten vragen extra bescherming</strong><p>Een ondernemersaccount bevat organisatie- en medewerkersgegevens. Gebruik daarom altijd een uniek wachtwoord en tweestapsverificatie.</p></div>
