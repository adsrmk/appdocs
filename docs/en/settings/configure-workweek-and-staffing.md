title: Configure the workweek and staffing target
description: Define working days, default hours and desired availability.
outline: deep
---

Organization workweek settings are used in time, attendance and leave calculations.

## Define working days

Select the days that normally count as working days for the organization. Employee-specific agreements can differ, but the organization setting provides the default.

## Set expected hours

Use realistic default hours. Review existing registrations after a change to make sure reports are interpreted using the correct period and settings.

## Set the staffing target

Choose the minimum desired team availability. The target helps identify periods where approved or requested leave may create understaffing.

::: warning Review before saving
Changes can affect future calculations and how managers interpret dashboards. Communicate material policy changes to the team.
:::
