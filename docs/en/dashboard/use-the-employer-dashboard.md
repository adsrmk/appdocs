The dashboard combines the current status of your team so you can quickly see where attention is needed.

## Understand the key figures

| Figure | Meaning |
| --- | --- |
| Total hours worked | All registered hours for the selected week. |
| Average per employee | The average registered effort per team member. |
| Attendance | The attendance percentage based on organization settings. |
| Leave this week | Leave hours that fall within the selected week. |

## Use weeks and filters

Select the correct week first. Then filter by status or role to review a smaller group. Clear the filters to return to the full overview.

## Follow up on actions

Notification counters show whether leave requests are waiting for review. The navigation also shows outstanding time registrations. Open the corresponding page to inspect the details.
