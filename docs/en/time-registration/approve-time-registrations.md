Time registrations are grouped by employee and week so you can review a complete period before making a decision.

## Review a submitted week

1. Open **Time registrations**.
2. Select a week and employee.
3. Compare registered hours with the expected workweek.
4. Review notes, leave and unusual differences.
5. Approve the week or return it for correction.

## Before approval

Check that working days are complete, leave is recorded consistently and corrections have been explained. Approval confirms that the submitted week is ready for further processing.

::: warning Return incorrect entries
Do not edit an unexplained registration on behalf of an employee. Return the week with a clear correction request so the audit trail remains understandable.
:::
