# Heurs App Docs

Publieke, tweetalige supportdocumentatie voor de Heurs workforce-app.

- Nederlands op de hoofdroute
- English onder `/en/`
- Lokale full-text search zonder externe index
- Gebouwd met VitePress en geschikt voor Netlify

## Ontwikkelen

```bash
npm install
npm run docs:dev
```

## Controleren

```bash
npm run docs:build
npm run docs:preview
```

De inhoud volgt de categorieën van de supportomgeving in de workforce-app: aan de slag, dashboard, medewerkers, urenregistraties, verlofbeheer en instellingen.
